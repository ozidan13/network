انتخاب الـ Root Bridge

The Switch with the Lowest BID Becomes Root

الـ BID (Bridge ID) = Priority (2 bytes) + MAC Address (6 bytes). أقل BID = الـ Root Bridge.

SwitchPriorityMACBIDالنتيجة
SW-A 32768 00:11:22:33:44:11 32768.00:11:22:33:44:11 👑 ROOT BRIDGE
SW-B 32768 00:11:22:33:44:22 32768.00:11:22:33:44:22 Non-Root
SW-C 32768 00:11:22:33:44:33 32768.00:11:22:33:44:33 Non-Root
💡 Priority Default = 32768. ممكن تـ Override الـ Root بتغيير Priority لأقل قيمة (مضاعفات 4096). الـ Network Engineer لازم يقرر مين الـ Root — ما يتركهوش للـ MAC الأصغر!

حالات Port في STP

Port States: Blocking → Listening → Learning → Forwarding

State 1
Blocking
لا بيبعت ولا بيستقبل Frames. بس بيسمع BPDUs فقط.
State 2
Listening
بيبعت ويستقبل BPDUs. لا يتعلم MAC ولا يبعت Data.
State 3
Learning
بيتعلم MAC Addresses. لا يبعت Data بعد.
State 4
Forwarding
✅ يبعت ويستقبل Data بشكل طبيعي. الـ Port شغّال.
State 5
Disabled
Port مغلق يدوياً بـ `shutdown`. لا يشارك في STP.
⏱️ STP (802.1D) Classic بياخد ~30–50 ثانية للـ Convergence. RSTP (802.1w) بياخد أقل من ثانية! الشبكات الحديثة كلها RSTP.
