MAC Addresses & Ethernet

عناوين MAC والإيثرنت

كل كارت شبكة مصنوع في العالم عنده عنوان 48-bit فريد. مليار جهاز — ومفيش اتنين زي بعض. كيف؟

تشريح الـ MAC Address

48 bits = OUI (24 bits) + NIC (24 bits)

الـ MAC Address = 48 bit مقسومة لجزئين: أول 3 bytes (OUI) للشركة المصنعة، والـ 3 bytes الثانية (NIC) للكارت نفسه.

اكتب MAC Address (مثال: AA:BB:CC:11:22:33) وشوف تقسيمه:
Bit خاصة في أول Byte:
Bit 0 (LSB): Unicast/Multicast
0 = Unicast (عنوان جهاز واحد). 1 = Multicast (مجموعة).
Bit 1: Global/Local
0 = OUI (من IEEE). 1 = Locally Administered (مخصص يدوياً).

CSMA/CD — نظام تنظيم الوصول

Carrier Sense Multiple Access / Collision Detection

في شبكات الـ Ethernet القديمة (Hub بدل Switch) — كل الأجهزة بتشارك نفس الـ Cable. لو اتنين بعتوا في نفس الوقت = Collision. الـ CSMA/CD البروتوكول اللي بيديرها:

CS
Carrier Sense: استمع للـ Medium — مفيش إشارة = فاضي = تقدر تبعت
MA
Multiple Access: أكتر من جهاز بيحاول في نفس الوقت على نفس الـ Wire
CD
Collision Detection: لو Collision → بعت Jam Signal → كل الكل وقفوا → Random Backoff → حاولوا تاني
💡 الـ Switch ألغى الـ Collision Domain لكل Port — CSMA/CD بقى تاريخياً للـ Hub. Full Duplex مع الـ Switch = مفيش Collision.
