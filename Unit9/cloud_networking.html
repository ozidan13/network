IaaS vs PaaS vs SaaS

🖥️

IaaS

Infrastructure as a Service. VMs، Storage، Networking. أنت إداري كل حاجة. AWS EC2, Azure VM, GCE.

⚙️

PaaS

Platform as a Service. Deploy Code مباشرة. Heroku, AWS Elastic Beanstalk, Google App Engine.

☁️

SaaS

Software as a Service. تستخدم بس. Gmail، Office 365، Salesforce، Zoom.

AWS VPC Architecture

Virtual Private Cloud — Networking in the Cloud

VPC: 10.0.0.0/16 (aws region: us-east-1)
Public Subnet: 10.0.1.0/24 (AZ-a) → Internet Gateway
├── EC2: Web Server (t3.medium) — Public IP
└── Load Balancer (ALB)
Private Subnet: 10.0.2.0/24 (AZ-a) → NAT Gateway
├── EC2: App Server — No Public IP
└── RDS: PostgreSQL
Security Groups: (Stateful Firewall per Instance)
└── sg-web: inbound 443 from 0.0.0.0/0
└── sg-db: inbound 5432 from sg-app only

💡 Security Groups = Stateful Firewall للـ Instance. NACLs = Stateless Firewall للـ Subnet. التصميم: Private للـ Databases وPublic للـ Web Servers فقط.

Hybrid Cloud Connectivity

Site-to-Site VPN

AWS Direct Connect
On-Prem ←Fiber→ AWS
✅ سرعة وموثوقية عالية
✅ Consistent Latency
📌 للـ Mission Critical Workloads
