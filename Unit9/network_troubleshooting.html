Network Troubleshooting

حل مشاكل الشبكة بمنهجية علمية

العشوائية في Troubleshooting بتضيع الوقت. الـ OSI Model بيديك خريطة منهجية — ابدأ من Layer 1 واطلع فوق أو من Layer 7 ونزّل لتحت.

منهجية OSI Bottom-Up

Start Physical — Work Your Way Up

🔴 المشكلة: موظف مش قادر يفتح أي موقع إنترنت.
L1
Physical Layer
الكابل موصول؟ LED بيومض؟ ping 127.0.0.1 (loopback)
L2
Data Link Layer
MAC Table؟ إيه حالة الـ Port في الـ Switch؟ `show mac address-table`
L3
Network Layer
ping Gateway، ping 8.8.8.8، show ip route — إيه الـ Route؟
L7
Application Layer
nslookup google.com — هل DNS بيشتغل؟ الـ Browser Settings؟
