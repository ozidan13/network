Network Protocols

مقدمة البروتوكولات

بدون قواعد مشتركة — Samsung ما تقدرش تتكلم مع iPhone. البروتوكولات هي اللغة المشتركة للأجهزة.

ايه هو البروتوكول؟

Protocol = Agreed Rules of Communication

دلوقتي وانت بتقرأ ده — موبايلك يتبع عشرات البروتوكولات في نفس الوقت. بس معظم الناس ما بيعرفوش إنها موجودة، لأنها شغالة في الخفاء.

البروتوكول ببساطة = مجموعة قواعد متفق عليها بين الأجهزة تحدد: إزاي الرسالة بتتشكّل، إمتى بتتبعت، ازاي بيتم التحقق منها، وإيه اللي يحصل لو فيه خطأ.

📜 Protocol = A set of rules governing how devices communicate. Without protocols, two devices can't understand each other — even if they're physically connected.

مثال واقعي: شيلة بايجة تتصل بـ WiFi Router — طبقة الـ Physical جسدياً متربطين. بس عشان يتبادلوا بيانات، لازم يتفقوا على:

📦 Format: الرسالة بتبدأ بإيه وتخلص بإيه (Frame Structure)

⏱️ Timing: إمتى كل جهاز بيتكلم (لمنع التعارض)

🔢 Encoding: الـ Bits بتتمثل ازاي (0 vs 1 كإشارات)

✅ Error Checking: إزاي نعرف لو البيانات وصلت سليمة

البروتوكولات الأساسية

The Protocols Every Network Engineer Must Know

HTTP / HTTPS
Layer 7 — Application
بروتوكول نقل الصفحات. HTTPS = HTTP + TLS (مشفر). كل موقع بتفتحه بيستخدمه.
📌 المتصفح، APIs، WebApps
DNS
Layer 7 — Application
دفتر عناوين الإنترنت. بيحول الاسم (google.com) لـ IP Address (142.250.185.46) عشان الـ Router يعرف يروح فين.
📌 كل اتصال بإنترنت بيبدأ بـ DNS Query
TCP
Layer 4 — Transport
Transmission Control Protocol. موثوق ومضمون — بيضمن وصول كل البيانات بدون فقدان وبالترتيب الصح.
📌 المواقع، الإيميل، تنزيل الملفات
UDP
Layer 4 — Transport
User Datagram Protocol. سريع لكن مفيش ضمان. مناسب لما السرعة أهم من الدقة — زي الـ Stream.
📌 Live Video، Gaming، DNS، VoIP
IP
Layer 3 — Network
Internet Protocol. بيعطي كل جهاز عنوان (IP Address) وبيقرر المسار اللي الـ Packets هتاخده بين الشبكات.
📌 كل حاجة على الإنترنت تستخدم IP
DHCP
Layer 7 — Application
Dynamic Host Configuration Protocol. بيدي لجهازك IP Address تلقائي لما تتوصل بالـ WiFi. من غيره كنت هتحدد الـ IP يدوياً.
📌 كل WiFi Router بيشغّل DHCP
FTP / SFTP
Layer 7 — Application
File Transfer Protocol. بروتوكول نقل الملفات بين الأجهزة. SFTP هو النسخة الآمنة (مشفرة).
📌 رفع الملفات على السيرفرات، Web Hosting
SSH
Layer 7 — Application
Secure Shell. بيخليك تتحكم في جهاز تاني عن بُعد بشكل مشفر. أساسي لإدارة السيرفرات.
📌 إدارة الـ Linux Servers، Network Devices
