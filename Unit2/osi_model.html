The OSI Model

نموذج الـ OSI — 7 طبقات

أشيع سؤال في أي Networking Interview: "ايه هو الـ OSI Model?" خليك تعرف الإجابة 100% بعد الدرس ده!

تخيل معايا... 📦

عايز تبعت بيتزا من مطعم في القاهرة لصاحبك في الإسكندرية. عندك نظام كامل سلسلة العمليات دي: الطاهي يعمل البيتزا → تتحط في علبة → تتحط في شنطة → يجي عليها كورير → يسلمها → صاحبك يفتح الشنطة → يفتح العلبة → ياكل.

الـ OSI Model بالظبط كده — 7 "طبقات" من العمليات بتحصل لما البيانات بتتنقل من جهازك لجهاز تاني. كل طبقة بتعمل شغلة محددة!

📖 OSI = Open Systems Interconnection — نظام مرجعي وضعته ISO سنة 1984 عشان كل الشركات تتكلم بنفس اللغة في الشبكات.

Memory Tricks to Remember the 7 Layers!

فيه اصطلاح شهير بالإنجليزي من فوق لتحت:

All — Application
Priests — Presentation
Seem — Session
To — Transport
Need — Network
Data — Data Link
Processing — Physical

أو بالعربي المصري: "أبو سمحان تنن دنا بالنا" 😄
(أبو = Application, سمحان = Session, تنن = Transport, دنا = Data Link, بالنا = Physical)

والأهم: حفظ الـ PDU لكل طبقة:

7,6,5 = Data 4 = Segment 3 = Packet 2 = Frame 1 = Bits
