حالات TCP (State Machine)

11 States — From CLOSED to ESTABLISHED

CLOSED
لا اتصال. البداية والنهاية.
LISTEN
Server ينتظر SYN.
SYN_SENT
Client بعت SYN ينتظر SYN-ACK.
SYN_RECV
Server استقبل SYN وبعت SYN-ACK.
ESTABLISHED
✅ اتصال ناجح. Data تتدفق.
FIN_WAIT_1/2
Client بعت FIN ينتظر إغلاق.
CLOSE_WAIT
Server استقبل FIN — جاهز يغلق.
TIME_WAIT
ينتظر 2×MSL قبل CLOSED حقيقي.
💡 TIME_WAIT = 2 × MSL (Maximum Segment Lifetime = 120s) = ينتظر 4 دقائق. عشان يضمن كل الـ Packets الضائعة ماتأثرتش على اتصال جديد.
