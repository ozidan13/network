NTP — مزامنة الوقت

Network Time Protocol — Why Microseconds Matter

لو الـ Cisco Router والـ Firewall عندهم وقت مختلف — الـ Logs بتبقى مفيش معنى. الـ Kerberos authentication بيفضل لو الوقت اختلف أكتر من 5 دقائق. Trading Systems بتخسر بالملايين لو الـ Timestamps غلط. NTP بيحل كل ده.

# Stratum hierarchy
Stratum 0: Atomic Clock / GPS
Stratum 1: Primary NTP Server (pool.ntp.org)
Stratum 2: Your Company NTP Server
Stratum 3+: Your Devices
# Config example (Cisco)
ntp server pool.ntp.org prefer
ntp update-calendar

SNMP Trap إيه الفرق بينه وبين GET؟

نفس الشيء
GET = Manager يسأل الـ Agent. Trap = Agent يبعت تنبيه للـ Manager تلقائياً عند حدث معين
Trap للإعداد فقط
GET يستخدم UDP والـ Trap TCP
