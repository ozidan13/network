Capstone Design

مشروع التخرج — تصميم شبكة شركة

الـ Scenario: تصميم شبكة شركة متوسطة الحجم (200 موظف، 3 مبانٍ، Data Center) من الصفر. هدفك: High Availability وSecurity وScalability.

Project Requirements

🏢 Requirements
200 موظف — 3 مبانٍ — Data Center مركزي — WAN Connectivity — WiFi لكل طابق — VOIP System.
🔒 Security Requirements
VLAN Segmentation — NGFW — IPS — VPN للـ Remote Workers — DMZ للـ Public Servers.
⚡ Availability
99.9% Uptime — Redundant Links — HSRP — Dual ISP — UPS — Generator.
📈 Scalability
تصميم يتحمل نمو حتى 500 موظف بدون إعادة تصميم كاملة. OSPF للـ Routing.

Task: خطوات الـ Implementation

1. IP Addressing Planning
خطط الـ Subnets والـ VLANs وقوم بـ Documentation قبل أي كونفيج.
2. Physical & Logical Topology
ارسم الـ Diagram بأجهزة Core/Distribution/Access وكل الـ Links والـ Redundancy.
3. Cisco Configuration (Lab)
استخدم Cisco Packet Tracer أو GNS3. Configure VLANs، OSPF، HSRP، ACLs.
4. Security Hardening
Disable Telnet، Enable SSH، Configure ACLs، Port Security، DHCP Snooping.
5. Testing & Documentation
اعمل Ping Tests، Failover Tests. اكتب Network Runbook وسجّل كل حاجة.
