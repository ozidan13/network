IPv6

الجيل الجديد من العناوين

IPv4 خلص رسمياً في 2011. اللي بيشيل الإنترنت دلوقتي = NAT + IPv6 مع بعض. والمستقبل كله IPv6.

الحجم — رقم لا يُتخيَّل

From 4.3 Billion to 340 Undecillion

IPv4 — 32 bit
4.3 مليار
2³² = 4,294,967,296 عنوان
❌ خلصوا رسمياً 2011
🔧 NAT كحل مؤقت
IPv6 — 128 bit
340 Undecillion
2¹²⁸ = 3.4 × 10³⁸
✅ لكل جهاز عنوان خاص
🚀 بلا NAT، بلا حدود
🌍 COSMIC SCALE

لو كل رملة على الأرض (7.5×10¹⁸) أخدت عنوان IPv4 → ما كفى. بـ IPv6 — كل رملة تقدر تاخد 340 مليون عنوان مليار مرة وكمان يفضل!

كتابة عناوين IPv6

128 bits = 8 groups × 16 bits (Hexadecimal)

العنوان الكامل: 8 مجموعات، كل مجموعة 4 hex digits، مفصولة بـ :

قواعد تبسيط الكتابة:
# Full:
2001:0DB8:0000:0000:0000:FF00:0042:8329
# Rule 1 — Remove leading zeros:
2001:DB8:0:0:0:FF00:42:8329
# Rule 2 — :: replaces longest run of 0s (once only):
2001:DB8::FF00:42:8329

أنواع عناوين IPv6

No More Broadcast — Multicast & Anycast Instead

Global Unicast
2000::/3
عناوين عامة = مثل IPv4 Public. تبدأ بـ 2 أو 3.
Link-Local
FE80::/10
للتواصل داخل نفس الـ Link فقط. تُعيَّن تلقائياً لكل interface.
Unique Local
FC00::/7
مثل IPv4 Private. للشبكات الداخلية. لا تُوجَّه للإنترنت.
Multicast
FF00::/8
لمجموعات من الأجهزة. يحل محل IPv4 Broadcast تماماً.
Anycast
من نفس Global
نفس العنوان على عدة أجهزة — Packet يروح لأقربهم. للـ DNS وCDN.
Loopback
::1/128
مثل 127.0.0.1 في IPv4. الجهاز يتكلم مع نفسه.

التحول من IPv4 لـ IPv6

Not Overnight — 3 Transition Mechanisms

🔀

Dual Stack

الجهاز بيشتغل بـ IPv4 وIPv6 في نفس الوقت. التطبيق يختار. الأشيع حالياً.

🚇

Tunneling

تغليف IPv6 Packet جوه IPv4 Packet لعبور شبكة IPv4. 6to4، Teredo.

🔄

NAT64

Gateway يترجم بين IPv6 وIPv4 — عشان IPv6-only clients يتكلموا مع IPv4 servers.
