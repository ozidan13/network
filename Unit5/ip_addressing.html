الـ IP Classes

Classful Addressing — A, B, C, D, E

قبل الـ CIDR (1993) — الـ IP Addresses كانت متقسمة لفئات (Classes). الـ First Octet بيحدد الـ Class تلقائياً:

Class A1—126128 Networks × 16M Hosts
Class B128—19116K Networks × 65K Hosts
Class C192—2232M Networks × 254 Hosts
Class D224—239Multicast
Class E240—255Reserved / Research
💡 الـ Classful Addressing انتهى بـ CIDR. دلوقتي الـ /24، /16، /8 بتحل محل Class A/B/C. بس لازم تعرف الـ Classes للامتحانات ولكتير من الـ Legacy configs.

IPv4 ضد IPv6

Why IPv6 — and Why It's Taking So Long

IPv4

الحجم: 32 bit → 4.3 مليار عنوان

الشكل: 192.168.1.1

الوضع: نفدت عناوينه رسمياً 2011

الحل المؤقت: NAT (مئة جهاز على IP واحد)

IPv6

الحجم: 128 bit → 340 undecillion عنوان!

الشكل: 2001:db8::1

الوضع: 45% من الإنترنت دلوقتي IPv6

ميزة: No NAT، Auto-config، IPSec مدمج

🌍 REAL SCALE

340 undecillion = 340 × 10³⁶. لو كل رملة على الأرض أخدت IPv4 — ما وفّتش. IPv6 يعطي كل رملة 340 تريليون عنوان. مش هينفد.
