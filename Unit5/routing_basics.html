📌 OSPF، EIGRP، BGP

بروتوكولات الـ Routing

IGP vs EGP — Inside the Network vs Between Networks

RIP v2

Distance Vector. يقيس المسافة بعدد الـ Hops. Max 15 hops. بطيء، قديم، للشبكات الصغيرة جداً.

OSPF

Link State. يبني خريطة كاملة للشبكة. Dijkstra Algorithm. الأشيع داخل الشركات الكبيرة.

EIGRP

Cisco proprietary. Hybrid (Distance Vector + Link State). سريع جداً في التقارب. للبيئات Cisco.

BGP

بروتوكول الإنترنت. بيشيل توجيه الـ Traffic بين الـ ISPs والدول. "بروتوكول البروتوكولات". Path Vector.
